# Руководство пользователя

Ниже описан основной сценарий: конвертация логов QGroundControl (.ulg) в ML-готовый CSV.

## Логи QGroundControl (.ulg) → CSV одной командой

1. Сохраните логи QGC в одну из папок: `qgc_logs/`, `~/qgc_logs/` или `~/QGroundControl/Logs`.
2. Запустите конвертацию выбранного лога (пример):
   ```bash
   python3 scripts/ulog_to_ml_csv.py /path/to/log.ulg --out runs
   ```
3. Итог появится в `runs/<имя_лога>/`: файл `flight_data.csv` и папка `csv_raw/` с исходными темами. Скрипт также выведет превью содержимого CSV (столбцы, типы, первые/последние строки, базовую статистику).

## Где искать результаты

- `runs/<timestamp или имя_лога>/` — артефакты конвертации (CSV и исходные CSV по темам).
